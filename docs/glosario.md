## Glosario de términos

**Utility (empresa eléctrica)**  
Empresa que provee servicio eléctrico en una región. En este proyecto representan posibles clientes o actores clave para prospecting.

**Subestación eléctrica**  
Instalación donde se transforma y distribuye energía eléctrica. Se utiliza como proxy de infraestructura y complejidad operativa.

**ZIP**  
Código postal de EE.UU. Se usa como unidad geográfica para el análisis y ranking.

**Ranking / Rank**  
Posición relativa de un ZIP en función de su puntaje de prioridad (1 = más prioritario).

**Puntaje de prioridad (score)**  
Valor normalizado entre 0 y 1 que combina presencia de utilities e infraestructura eléctrica.

**IOU (Investor-Owned Utility)**  
Utility privada con fines de lucro.

**Non-IOU**  
Utility cooperativa, municipal u otro modelo no privado.

**Cobertura de ZIPs prioritarios**  
Cantidad de ZIPs del Top N donde una utility tiene presencia.

**Voltaje (kV)**  
Nivel de tensión eléctrica. Voltajes altos indican infraestructura más compleja.

**Tipo de infraestructura**  
Clasificación del activo eléctrico (subestación, tap, riser, etc.).

**Estado operativo**  
Condición del activo (en servicio, en construcción, no disponible).
